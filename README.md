# EsoDaTres

<h2>Informed/Uninformed Search miniproject </h2>
<p>
The purpose of this minigame is to show the differences between search techniques such as uniform cost and A* within an easy to understand environment.

<h2> About the game </h2>
<p>
Each technique is shown with a path taken by the subject in order to reach the goals, there are penalizations and bonus in the evironment that increment or decrement the cost of movement through certain areas and allow to understand easily the priorities among the different types of search.
